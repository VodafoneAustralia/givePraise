# Praise Me Application
## Details
An application to praise fellow staff members
### Running the app
```
Clone The repo:
$ git clone git@github.com:thepig25/givePraise.git
Install Meteor
$ curl https://install.meteor.com/ | sh

Run The app
$ cd give-praise
$ meteor
Open
 http://localhost:3000
 

```
